Скрипт который 
1. Помогает избавиться от большей части рутинной работы в создании журнала
2. Обрабатывает форму статьи из excel документа в word документ

Для начала работы предстоит установить некоторые библиотеки из файла requirements.txt
Это можно сделать, вставив в командную строку/терминал ```pip install -r requirements.txt``` 

После, можно запустить код из section.py и затем этого создаться файл conference_sections.docx в котором будут последовательно и отсортированы по секциям находится статьи
